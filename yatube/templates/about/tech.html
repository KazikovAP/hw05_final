Технологии применимые для создания проекта SocialNetwork

  • Python - язык программирования на котором написан сам проект.
  • Django - высокоуровневый веб-фреймворк предназаначенный для создания веб-приложений.
  • Django ORM - инструмент, предоставляемый Django, обеспечивающий удобный и абстрактный способ взаимодействия с базами данных.
  • GitHub - платформа для хостинга и совместной работы с репозиториями Git.
  • Git - распределенная система контроля версий, которая используется для управления изменениями в исходном коде и других файлах проекта.
  • HTML - язык разметки, используемый для создания и структурирования веб-страниц.
  • CSS - язык стилей, который используется для оформления и визуального представления веб-страниц.
  • Bootstrap - популярный фреймворк (набор инструментов) для разработки веб-интерфейсов.
  • Unittest - модуль в стандартной библиотеке языка Python, который предоставляет инструменты для создания и запуска модульных тестов.
  • Pythonanywhere - платформа веб-хостинга, которая позволяет разработчикам запускать и развертывать веб-приложения, написанные на языке программирования Python.
