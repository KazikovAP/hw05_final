Этот проект создан при помощи Django фреймоврка с простенькой СУБД SQLite3 "из коробки". В качестве CSS использовался Bootstrap. Здесь можно выкладывать посты, редактировать их, добавлять картинки, оставлять комментарии к постам. А так же подписываться на других авторов. Регистироваться, менять пароли (хотя на хостинге может пока не рабоать) В общем реализован базовый функционал соц. сети. Не судите строго за внешний вид. По ходу изучения html и css буду стараться улучшать внешний вид проекта.
